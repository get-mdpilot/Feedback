# MDPilot — Feedback & Bug Reports

This is the public feedback home for [MDPilot](https://mdpilot.in) — the markdown intelligence platform — and the [`mdpilot-mcp`](https://www.npmjs.com/package/mdpilot-mcp) MCP server.

## Report something

| | |
|---|---|
| 🐛 **Something broke** | [File a bug report](https://github.com/get-mdpilot/Feedback/issues/new?template=bug_report.yml) |
| 💡 **Idea or improvement** | [Share feedback](https://github.com/get-mdpilot/Feedback/issues/new?template=feedback.yml) |
| ✉️ **Anything else** | [privacy@mdpilot.in](mailto:privacy@mdpilot.in) |

Both templates take under two minutes. The more specific you are (steps, surface, MCP client, version), the faster we can fix it.

## What is MDPilot?

- **Task mode** — paste any ticket or thread, get a precise, gap-checked TASK.md for your AI agent
- **Generate** — production-grade AGENTS.md, CLAUDE.md, README and 6 more file types, grounded in your real repo
- **MCP server** — all of it inside Claude Code, Cursor, Windsurf, or Goose: `npx -y mdpilot-mcp setup`
- **Token optimizer** — 5-pass compression, 20–40% smaller instruction files
- **Drift detection** — finds docs that no longer match your repo
